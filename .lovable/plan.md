
## Birthday Greeting Website

A beautiful, emotional birthday greeting page with pastel aesthetics, glassmorphism, and smooth animations.

### Design & Layout
- Soft pastel gradient background (pink → lavender → white)
- Floating hearts 💖 and sparkles ✨ animated in background
- Glassmorphism card (backdrop-blur, semi-transparent, soft border)
- Centered, mobile-first responsive layout
- Google Fonts: "Poppins" (body) + "Dancing Script" (headings)

### Sections
1. **Hero** — Large glowing "Happy Birthday 😊" heading in Dancing Script
2. **Photo** — Circular placeholder image with glowing border, soft shadow, fade-in + zoom animation
3. **Message** — The full birthday message with fade-in/typing effect, including the playful "best friend" strikethrough moment
4. **Gift Button** — Rounded "Click here 🎁" button with glow + hover animation; on click reveals hidden message with fade+scale
5. **Extra Lines** — Additional kind words with fade-in

### Animations & Effects
- CSS keyframe floating hearts/sparkles (15-20 elements, randomized positions/speeds)
- Fade-in on scroll for each section
- Gentle zoom on photo load
- Button glow pulse on hover
- Smooth reveal animation for hidden message
- Dreamy glowing particle dots via CSS

### Extras
- Background music toggle (mute/unmute button) — will use a placeholder; user can add their own audio file
- Smooth scrolling enabled
- Fully responsive with proper mobile spacing
